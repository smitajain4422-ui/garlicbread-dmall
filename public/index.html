Comprehensive Guide

Mastering the Nosify DMall Engine.

1. Managing Tokens & Profiles

Navigate to the Config tab. Paste your Discord bot tokens here. You can group them into "Folders" (e.g., "Main Server", "Backup Bots"). Always click Check after adding them to ensure they aren't terminated by Discord.

Customizing Bots: In the Token table, click the Edit button next to an active bot. You can instantly change the bot's Username and Avatar (Profile Picture) across Discord to make them look authentic.

2. Creating & Previewing Embeds

In the Config tab, use the Embed Builder. You must paste raw, valid Discord JSON (from generators like Discohook). Click Preview Layout to see exactly how it will look inside Discord before you send it. Name it something memorable and hit Save.

3. Delays & Avoiding Rate Limits

Discord will ban bots that send DMs too quickly. In the Config tab, adjust your Matrix variables:
- Delay (ms): How long the engine pauses between each individual DM. Default is 200ms. Increase to 500-1000ms if your bots are getting 403 Forbidden errors.
- Concurrency: How many bots attempt to send a DM at the exact same time. Default is 20.

4. Fetching Servers & Launching

Go to the Start (Home) tab. Select your Bot Folder. The system will automatically reach out to Discord using your bot tokens and fetch a dropdown list of every server those bots are currently sitting in. Select the server, choose your audience (Online vs All), select your saved Embed, and click Start. You will be pulled into the Live Console to watch the data stream.

5. Blacklisting & Resetting Logs

Don't want to accidentally DM an admin? Put their Discord User ID into the Blacklist in the Config tab. The engine will instantly skip them.

Re-Running Campaigns: The engine safely remembers who it has DMed so it doesn't spam them twice. To run a fresh campaign on the exact same server, click the red Reset Logs button in the Config tab.
