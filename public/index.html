Comprehensive Guide

Mastering the Nosify DMall Engine.

1. Memory & Resumption

Because this site runs on cloud servers, it will pause if you close your browser tab. However, progress is saved per-server. If you accidentally close it, just select the server again in the Start tab. It will detect the paused memory and automatically resume exactly where it left off! You can also wipe this memory manually.

2. Scraping 10k+ Members

The system is built to scrape massive servers. Discord will periodically rate-limit the bots during scraping. Do not panic! The engine will automatically pause for the exact duration Discord requests, and resume scraping until 100% of the members are fetched.

3. Mass Editing & Tokens

Paste arrays of tokens and hit Check All Bots. Use Mass Edit Profiles to change the names and avatars of every alive bot in your database at once. Use Copy All Invites to instantly pull them into your server.

4. Embed Variables

Personalize DMs in the Embed Builder:
- {userid} : Target's raw Discord ID.
- {usermention} : Actual ping/mention (e.g. @User).
